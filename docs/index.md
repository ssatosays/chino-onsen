### 4. おすすめ

#### 4-1. 白樺湖温泉すずらんの湯

建物自体は年季が入ってる感じだが、広くて清掃も行き届いてるイメージ。行楽シーズンの紅葉やスキーの時期を外せば、わりと空いているはず。
サウナが低温(60°)なのが、いいのか悪いのか分からんがゆっくりできる。
施設職員も好印象。

### 4-2. 小斉の湯

かなり秘湯的な温泉。多分、源泉かけ流し系。基本激熱。
12-2月ごろの夜とかに行くと、-10°前後?(か、もっと低い)で露天風呂でも、かなり寒暖差できるため軽く「ととのう」気がする。
施設職員/客層ともにノーコメント。
